## 🌤 ESP32 OLED Weather / System Display PCB

A compact environmental monitoring and system display PCB built around the Arduino Nano ESP32. The board integrates a 0.91-inch I²C OLED display and a Bosch BME280 environmental sensor for measuring temperature, humidity, and atmospheric pressure. Designed in KiCad with an emphasis on clean routing, dual-layer ground planes, reliable I²C communication, and successful ERC/DRC verification.

**Features**
- Arduino Nano ESP32
- Bosch BME280 Environmental Sensor
- 0.91-inch I²C OLED Display
- USB-C Power Input
- Dual-Layer Ground Planes
- I²C Communication
- ERC: 0 Errors, 0 Warnings
- DRC: 0 Errors, 0 Warnings
