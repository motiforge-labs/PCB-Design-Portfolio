## ⌨️ ESP32 Macro Pad Pro PCB

A compact 6-key programmable macro pad built around the Arduino Nano ESP32. The design integrates WS2812B RGB LEDs for per-key lighting, a 0.91-inch I²C OLED display for status and menu information, USB-C connectivity, and a dedicated reset button. The PCB was designed in KiCad with a focus on clean component placement, efficient routing, dual-layer ground planes, and successful ERC/DRC verification.

**Features**
- Arduino Nano ESP32
- 6 Mechanical Key Switches
- 6 × WS2812B RGB LEDs
- 0.91-inch I²C OLED Display
- USB-C Connectivity
- Dual-Layer PCB
- ERC: 0 Errors, 0 Warnings
- DRC: 0 Errors, 0 Warnings
