## 🌈 RGB Mood Light PCB

A compact RGB mood lighting PCB built around the Arduino Nano ESP32. The board drives multiple WS2812B addressable RGB LEDs while being powered through USB-C. Designed in KiCad, the project demonstrates clean schematic design, efficient PCB routing, proper power distribution, and successful ERC/DRC verification.

**Features**
- Arduino Nano ESP32
- WS2812B Addressable RGB LEDs
- USB-C Power Input
- Dual-Layer PCB
- Clean Power Distribution
- ERC: 0 Errors, 0 Warnings
- DRC: 0 Errors, 0 Warnings
